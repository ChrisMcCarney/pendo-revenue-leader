## Pendo positioning (FY27)

Pendo's product line-up under the FY27 narrative is structured around five named pillars - **Listen**, **Orchestrate**, **Predict**, **Agent Analytics**, **Data Sync** - sitting over the existing capability set (Analytics, Guides, Session Replay, Feedback / NPS, Roadmap). Use these pillar names in competitive framing rather than the older capability-only language.

| FY27 pillar | What it covers | Beats / parity vs competitors |
| --- | --- | --- |
| **Listen** | NPS, sentiment, in-app feedback, surveys, Sentiment AI | Beats Qualtrics on time-to-signal; parity with Sprig; native to the product surface vs survey-only tools |
| **Orchestrate** | Guides, in-app messaging, onboarding flows, walkthroughs | Beats WalkMe on auto-capture-driven targeting; beats Appcues on enterprise scale; parity with Userflow |
| **Predict** | AI features: forecasting, recommendations, agent suggestions | Differentiator vs Amplitude / Mixpanel - they ship analytics; Pendo ships analytics + action |
| **Agent Analytics** | Rage-prompt detection, agent retention, AI-product telemetry | New category; no direct competitor; LangSmith / Helicone overlap on a narrow slice |
| **Data Sync** | Warehouse sync (Snowflake, BigQuery, Databricks), reverse-ETL, integrations | Parity with Hightouch / Census on direction; Pendo's edge is having the source data already |

When framing against per-competitor files, lead with the pillar Pendo wins on, not the capability they overlap on.
