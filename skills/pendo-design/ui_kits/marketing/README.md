# Pendo Marketing UI Kit

A marketing landing-page kit built from the FY27 brand spec. Not a recreation
of a specific Pendo marketing page — those were not provided — but a
realistic composition of a Pendo-style hero, feature grid, stats panel,
customer logos strip, quote block, and CTA footer using the brand's
visual foundations.

## Components

- `NavBar.jsx` — fixed dark nav with chevron logo + links + pink CTA button
- `Hero.jsx` — dark hero with pink eyebrow + Sora headline + pink rule + CTAs
- `FeatureGrid.jsx` — three graphite cards, pink top-bars, eyebrow labels
- `StatPanel.jsx` — dark gradient, three Sora ExtraBold stats with pink bolts
- `LogosStrip.jsx` — horizontal customer-logo row on cream
- `QuoteBlock.jsx` — oversized opening quote mark + bold Sora quote + headshot
- `CTAFooter.jsx` — closing band, pink, with large CTA

## Open

`index.html` assembles them into a full page.
